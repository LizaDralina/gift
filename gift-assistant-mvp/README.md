# Gift Assistant MVP

Backend MVP для курсового проекта "Приложение помощник для выбора подарка".

## Запуск

### 1. Создать и активировать виртуальное окружение
python -m venv venv

Linux/macOS:
source venv/bin/activate

### 2. Установить зависимости
pip install -r requirements.txt

### 3. Запустить сервер
uvicorn app.main:app --reload

## Swagger
После запуска документация будет доступна по адресу:
http://127.0.0.1:8000/docs

## Основные возможности
- регистрация и авторизация пользователя
- JWT-аутентификация
- создание и редактирование профиля получателя
- импорт товаров из CSV
- просмотр каталога
- генерация рекомендаций с объяснениями

## База данных
По умолчанию используется SQLite:
gift_mvp.db
